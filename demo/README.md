# Demo Materials

This folder contains presentation-ready materials for pitching HireProof.

## Files

- `problem-statement.md`: concise problem framing, user pain, urgency, and opportunity.
- `pitch.md`: short, one-minute, and five-minute pitch scripts.
- `slides.md`: Markdown slide deck outline for presenting the idea.
- `slide-wireframe.md`: visual layout blueprint for turning the slide outline into a polished deck.
- `research-notes.md`: sourced research notes supporting the demo narrative and technical framing.

## Core Message

HireProof helps job seekers check suspicious job posts before applying. It turns a risky, manual verification process into a clear investigation with proof, risk scoring, and safer alternatives.
