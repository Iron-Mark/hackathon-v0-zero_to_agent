# Make Custom App Submission Checklist

- [ ] Create a Make developer app named `HireProof`.
- [ ] Add the `HireProof API` connection with API key header auth.
- [ ] Add `Audit job post`, `Audit job post async`, and `Get API health` modules.
- [ ] Confirm mappable text/location/webhook fields.
- [ ] Run the demo fixture with `hireproof_agent_demo_key`.
- [ ] Capture screenshots for connection setup, sync audit, async audit, and health check.
- [ ] Submit for Make review.
- [ ] Do not claim public Make approval until Make approves the app.
